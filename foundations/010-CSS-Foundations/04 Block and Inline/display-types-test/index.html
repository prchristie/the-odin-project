Block

This is a block level element. It will be on a new line. Its width should be that of its parent content. The parent element here has padding, so it fill up its entire content

Even if the content is too small, its still wide

The paragraph here is a block! But this is an inline element here. It doesn't change anything (other than the colour), because inline just chills inline.

In this one, we will see that the inline element has some padding and margin.This is respected in the inline direction (horizontal), but not the block direction (vertical).

We could fix this "problem" and make everything respect us by changing our inline element to be an inline block. There is no padding here, but this is a block element that is also inline

Now lets see what happens when we put some margin and padding.. Oh shit, we got some respect, exactly like a block element would expect. Das pretty cool. But we don't force a new line. Except we kind of have?
